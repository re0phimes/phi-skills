# Usage Examples

## Frontend Feature Flow

Prompt:

`phi-frontend tighten the homepage header, preserve dark mode, and keep mobile behavior consistent`

Expected orchestration:

1. `phi-brainstorming` owns design clarification
2. ECC `frontend-patterns` supports the implementation design
3. `phi-planning` produces the execution plan
4. `phi-execute` drives implementation
5. `phi-verify` performs completion checks

## Debugging Flow

Prompt:

`phi-debugging investigate why the production build fails after the new API route change`

Expected orchestration:

1. `superpowers` debugging discipline leads the process
2. ECC domain guidance is selected from the relevant stack
3. `phi-verify` confirms the fix before handoff

## FAQ Flow

Prompt:

`phi-faq generate QA pairs from this markdown file and review the best candidates`

Expected orchestration:

1. local FAQ skills perform generation and review
2. `phi-verify` confirms the output is ready for use
